# Raspberry-BiometricSensor
Raspberry Pi biometric sensor (oximeter, LED display). This project reads real-time data like heart rate and oxygen levels and shows it on the screen. It tests the core technology of health sensors and microcomputers working together.

## Components Used
* Raspberry Pi
* Breadboard
* Jumper wires
* Oximeter sensor
* LED display

## How to Replicate
If you want to understand how this system works, the best way is to try it yourself. 
1. Download the code from this repository.
2. Wire the components to the Raspberry Pi pins.
3. Run the code on your Raspberry Pi.
4. Place your finger on the oximeter to test the sensor and see the results on the display.
